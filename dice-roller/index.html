Dice Roller

Each role-playing game needs its own dice set, so each character sheet is able to give direct access to the dice roller.
Use the character sheet menu to access it, or use the three finger tap on any page.
It consists of two sections, accessible via the swipe gesture:

  • The calculator

    Like a common calculator, just write the complete expression to get the result.
    A simple example could be one 6-sided dice, to which add the value 1

    A more complex example could be roll two 20-sided dice, add 4 and multiply all by the result of one 3-sided dice.

    In the part below, next to the result you will see the whole expression with all the results of the dice rolls highlighted

  • The formula collector

    It often happens that the same roll of the dice is repeated several times during the games, thanks to the formula collector you can collect each written expression, give it a name, and reuse it whenever you want.
